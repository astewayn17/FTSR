backend/ - The responsibility of this directory is to provide the application with a Spring Boot based backend to facilitate the basic function of the application.



frontend/ - The responsibility of this directory is to provide the application with an Angular focused frontend to provide the visual UI and UX of the application.



infra/ - The responsibility of this directory is to provide the application with a set of Jenkins, Docker, Sonar. etc. pipeline tools to provide seamless CI and CD of the application.
